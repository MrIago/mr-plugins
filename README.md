# mr-plugins

Claude Code plugin marketplace by [mriago](https://github.com/MrIago).

## Installation

```bash
/plugin marketplace add MrIago/mr-plugins
```

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [check-rules](./check-rules/) | Audit files against project rules with parallel batch processing | `/plugin install check-rules@mr-plugins` |
