# Claude Code Plugin

Claude Code CLI and SDK reference for programmatic use.

## Features

- **Headless Mode** - Programmatic execution with `-p` flag
- **Sandbox SDK** - Build isolated agent environments
- **Hooks & Plugins** - Extend Claude Code functionality
- **MCP Servers** - Model Context Protocol integration
- **CI/CD Integration** - GitHub Actions, GitLab CI/CD

## Installation

```bash
/plugin marketplace add MrIago/mr-plugins
/plugin install cc@mr-plugins
```

## Usage

The skill is invoked automatically when you ask about Claude Code topics, or manually:

```bash
/cc <question about Claude Code>
```

## Topics Covered

- CLI Reference & Commands
- Headless Mode & Output Formats
- Skills, Sub-agents, Hooks
- Plugins & Marketplaces
- MCP & LSP Servers
- IDE Integrations (VS Code, JetBrains)
- Cloud Deployments (Cloudflare, AWS, GCP)
- Security & IAM
