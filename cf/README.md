# Cloudflare Plugin

Comprehensive Cloudflare platform development reference.

## Features

- **Workers** - Serverless functions, bindings, cron triggers
- **Durable Objects** - Stateful actors, WebSockets, SQLite
- **Queues** - Message queues, background jobs, batching
- **Workflows** - Durable execution, steps, retries
- **Hyperdrive** - PostgreSQL/MySQL connection pooling
- **Workers AI** - LLM inference, embeddings
- **Sandbox** - Code execution, Claude Code integration
- **Containers** - Long-running processes, full Linux

## Installation

```bash
/plugin marketplace add MrIago/mr-plugins
/plugin install cf@mr-plugins
```

## Usage

The skill is invoked automatically when working with Cloudflare, or manually:

```bash
/cf <question about Cloudflare>
```
